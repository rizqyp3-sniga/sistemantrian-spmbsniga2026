# Sistem Antrian SPMB SMP Negeri 3 Pemalang

Aplikasi web untuk manajemen antrian penerimaan siswa baru.

## Fitur
- Antrian otomatis dibuka pukul 19:00
- Kuota terbatas 300 per hari
- Simpan data ke Google Sheets otomatis
- Mode Light/Dark Theme
- Responsive dan mobile-friendly

## Cara Menggunakan

1. Buka link: https://nama-anda.github.io/antrian-spmb
2. Klik tombol "Ambil Antrian" (hanya tersedia jam 19:00)
3. Isi data pendaftaran
4. Nomor antrian otomatis tersimpan di Google Sheets

## Webhook Google Sheets
Data sudah terhubung dengan Google Apps Script untuk penyimpanan otomatis.
